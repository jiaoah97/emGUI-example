## Keyboard shortcuts

* **ESC** - similar to close button in StatusBar

## How to build

Checkout project

```
git clone git@github.com:libEmGUI/emGUI-example.git
git submodule init
git submodule update
```

Use MS Visual Studio 2015 to build `emGUI-example/emGUI-example.sln`
